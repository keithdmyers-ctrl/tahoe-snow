# Deploy to Hugging Face

1. Run local tests to verify the app works
2. Check for any syntax errors or import issues
3. Git add, commit, and push to GitHub
4. Push to Hugging Face Space (keithdmyers/tahoe-snow or configured remote)
5. Wait for the Space to build and verify it's running
6. Report deployment status
